# Eevee Evolution Guide

## Project Information
The Eevee Evolution Guide is an app that helps users learn about Eevee and all of its evolutions. Instead of searching online, users can choose an evolution and see information about it in one place.

## What Problem Does the App Solve?
Sometimes it's hard to remember all of Eevee's evolutions, their types, or how they evolve. This app makes it easier by putting all of the information together in a simple program.

## Who Is the App For?
This app is for Pokémon fans, especially people who like Eevee or want to learn more about its evolutions. It can also be helpful for new players who are still learning about Pokémon.

## Main Features
- View all of Eevee's evolutions.
- Select an evolution to learn more about it.
- Display information such as the Pokémon's type and how it evolves.
- Easy-to-use menu for navigating the app.

## How the App Works
When the program starts, the user is shown a list of Eevee's evolutions. The user chooses one from the menu, and the program displays information about that evolution. The user can continue looking at different evolutions until they decide to exit the program.
