## User Interface 
The User Interface for the Eevee Evolution Guide will be very simple and easy to use. When the application starts up, the user will see a welcome message and a cute menu showing Eevee and it's different evolution's. 

The user would then select a Pokémon by entering the number next to it's name! After the user makes a selection, the app will display information about that Pokémon. Information included is: it's type, how it evolves, and a short description. 

The user then will be able to return to the main menu and select another evolution. The menu will also include an option to exit the application.
